# Flashcard
Flashcard App is a simple and interactive web-based learning tool built using HTML, CSS, and JavaScript. It allows users to create custom flashcards containing questions and answers. Users can click on a card to flip it and reveal the answer with a smooth 3D animation effect.
An interactive Flashcard Web Application built using HTML, CSS, and JavaScript.
Create, flip, and delete flashcards easily with persistent storage using Local Storage.


🚀 Features

- ➕ Add new flashcards (Question & Answer)
- 🔄 Flip cards with 3D animation
- ❌ Delete flashcards
- 💾 Data saved using Local Storage
- 📱 Responsive layout
- 🎨 Modern UI design

🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Transitions, 3D Transform)
- JavaScript (DOM Manipulation, Local Storage)

🧠 How It Works

- User enters a question and answer.
- Clicking Add Card stores the data in an array.
- The data is saved to Local Storage.
- Cards are dynamically rendered using JavaScript.
- Clicking a card toggles a flip animation to reveal the answer.
